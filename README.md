# Software Engineering Portfolio

A simple, responsive portfolio website for a Software Engineering student looking for internship or junior developer roles.

## Sections

- Home
- About Me
- Skills
- Projects
- Contact

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages

## How to Edit

Open `index.html` and replace:

- `Your Name`
- `yourusername`
- `youremail@example.com`
- Project titles, descriptions, live demo links, and GitHub repo links

## How to Run Locally

Open `index.html` in your browser.

## How to Deploy on GitHub Pages

1. Create a GitHub repository named `portfolio` or `yourusername.github.io`.
2. Upload/push these files to the repository.
3. Go to repository `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.

Your site will be available at one of these links:

- `https://yourusername.github.io/portfolio`
- `https://yourusername.github.io`

## Git Commands

```bash
git init
git add .
git commit -m "Initial portfolio website"
git branch -M main
git remote add origin https://github.com/yourusername/portfolio.git
git push -u origin main
```
